# MLIDE 

A lightweight, high-performance code editor built with **Rust** and **egui**. Designed for speed and simplicity.
**And a little crooked**

## Features

* **Syntax Highlighting:** Support for Rust, C++, C#, and more.
* **Markdown Preview:** Real-time rendering for your documentation.
* **Image Viewer:** Integrated support for common image formats (PNG, JPG, etc.).
* **Integrated Terminal:** Run commands directly from the editor.
* **Smart Shortcuts:** Familiar hotkeys for line duplication, comments, and saving.

## Hotkeys

| Key | Action |
| :--- | :--- |
| `Ctrl + S` | Save current file |
| `Ctrl + /` | Toggle line comment |
| `Ctrl + D` | Duplicate current line |
| `Alt + ↑ / ↓` | Move line up or down |
| `Ctrl + B` | Toggle File Explorer |
| `Ctrl + Backtick` | Toggle Terminal |
| `Ctrl + W` | Close current tab |
